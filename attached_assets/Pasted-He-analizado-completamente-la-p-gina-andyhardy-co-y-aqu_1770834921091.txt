🔧 CÓMO SE IMPLEMENTA EL PARALLAX
Técnica Principal: jQuery + CSS Transform3D
La página usa una técnica muy elegante y eficiente basada en jQuery que manipula la posición Y de elementos al hacer scroll.
Archivo: parallax.js (El corazón del efecto)
javascript$(window).scroll(function() {
    var scrollTop = $(window).scrollTop();
    $('.parallax-layer').each(function() {
        var layer = $(this);
        var dataSpeed = layer.data('speed');
        var offsetY = -(scrollTop * dataSpeed);
        var translate = 'translate3d(0, ' + offsetY + 'px, 0)';
        
        layer.css('-webkit-transform', translate);
        layer.css('-moz-transform', translate);
        layer.css('-ms-transform', translate);
        layer.css('-o-transform', translate);
        layer.css('transform', translate);
    })
});
¿Cómo funciona?

Detecta el scroll del usuario con $(window).scrollTop()
Busca todos los elementos con clase .parallax-layer
Lee el atributo data-speed de cada elemento (velocidad de parallax)
Calcula el desplazamiento: offsetY = -(scrollTop * dataSpeed)
Aplica una transformación CSS 3D para mover el elemento

Las clases fade-in y appear controlan animaciones de entrada

Paso 4: Optimizaciones
Para mejor rendimiento, puedes añadir throttling:
javascriptvar ticking = false;

$(window).scroll(function() {
    if (!ticking) {
        window.requestAnimationFrame(function() {
            updateParallax();
            ticking = false;
        });
        ticking = true;
    }
});

function updateParallax() {
    var scrollTop = $(window).scrollTop();
    $('.parallax-layer').each(function() {
        var layer = $(this);
        var dataSpeed = layer.data('speed');
        var offsetY = -(scrollTop * dataSpeed);
        layer.css('transform', 'translate3d(0, ' + offsetY + 'px, 0)');
    });
}

Comienzan con opacity: 0 y top: 100px
Se animan a opacity: 1 y top: 0 con delay de 1.5s
Usa cubic-bezier para movimiento natural

2. Gradientes Estratégicos

Bottom fade: Oscurece hacia abajo (foco en el contenido)
